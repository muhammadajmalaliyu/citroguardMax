# 🍊 CITROGUARD MAX Landing Page

A modern, responsive landing page for **CITROGUARD MAX**, a natural pest repellent spray made with citrus extracts and herbs.

## Features

- Responsive design for desktop and mobile
- Premium modern UI
- Animated hero section
- Product showcase
- Ingredients section
- Pricing section
- Customer testimonials
- Contact & WhatsApp call-to-action
- Pure HTML & CSS (No JavaScript required)

## Technologies Used

- HTML5
- CSS3
- Google Fonts
  - Playfair Display
  - Inter

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/citroguard-max.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or build process is required.
